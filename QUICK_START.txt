═══════════════════════════════════════════════════════════════
  INSTALLATION
═══════════════════════════════════════════════════════════════

1️⃣  EXTRAIRE LE ZIP
   Mac/Linux : unzip insighter_ai_v1.0.zip
   Windows   : Click droit → Extraire

2️⃣  OUVRIR UN TERMINAL DANS LE DOSSIER

3️⃣  LANCER LE SERVEUR
   Mac/Linux : chmod +x start.sh && ./start.sh
   Windows   : Double-click start.bat

4️⃣  OUVRIR NAVIGATEUR
   http://localhost:5002

5️⃣  AJOUTER TA CLÉ API
   (Attention: ajoute-la avant de cliquer "Chercher")

═══════════════════════════════════════════════════════════════
  SI BESOIN DE LA CLE API
═══════════════════════════════════════════════════════════════

1. Va sur https://console.anthropic.com
2. Crée une clé API (gratuite)
3. Avant de lancer start.sh / start.bat, fais :

   Mac/Linux : export ANTHROPIC_API_KEY='sk-ant-...'
   Windows   : set ANTHROPIC_API_KEY=sk-ant-...

═══════════════════════════════════════════════════════════════
  ÇA NE MARCHE PAS ?
═══════════════════════════════════════════════════════════════

❌ "Python not found"
   → Installe Python : https://www.python.org

❌ "Port 5002 already in use"
   → Un autre service utilise ce port
   → Modifie app.py, ligne finale : port=5003

❌ "ANTHROPIC_API_KEY not found"
   → Définis la clé avant de lancer
   → export ANTHROPIC_API_KEY='...' (Mac/Linux)
   → set ANTHROPIC_API_KEY=... (Windows)
